# CekDown
Tools Check Site Down Or Normal (Untuk Mengecek Web Down Atau Normal)
![screenshot](./ss.jpg)

### Installation
```
pkg update && pkg upgrade
pkg install git python
pip install bs4 requests colorama
git clone https://github.com/Yutixcode/Cubdomain
cd Cubdomain
python cundomain.py
```
